Using events

Bind a function to the afterSet event to write "output". See the source →

OUTPUT:

JavaScript

var $oriel = $('ul').oriel({});
$oriel.bind('afterSet', function(e) {
  var oriel = e.oriel;
  var currentImage = oriel.currentIndex;
  $('#output').text(currentImage);
});

Events triggered:

beforeSet
afterSet
beforeInit
afterInit
beforePrev
afterPrev
beforeNext
afterNext
