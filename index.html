sshow.js

simple javascript slideshow jquery extension

A simple demo.

Default options are used with only minimal styling in CSS. See the source →

HTML

<ul>
  <li>
    <img src="images/leaf.jpg">
  </li>
  <li>
    <img src="images/leaves.jpg">
  </li>
</ul>

JavaScript

$('ul').sshow();

CSS

.sshow_stage { /* Should always set a height. */
  height: 600px;
}
.sshow_navigation { /* Style the navigation bar. */
  padding: 0.2em;
  background-color: black;
  color: #aaa;
}
.sshow_navigation a {
  color: #ccc;
}

Customizing.

The sshow object can control the slideshow programmatically.

For example: 

var el = $('ul').sshow(); // Create the slideshow
var slideshow = el.data('sshow'); // Obtain the slideshow object

slideshow.next(); // Go to next slide
slideshow.set(5); // Go to the 6th slide (0-indexed)
