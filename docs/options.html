options.

Pass in onImageChange function to write "output". See the source →

OUTPUT:

JavaScript

$('ul').oriel({
  onImageChange : function ( image ) {
    var output = $( '#output' );
    var href = $( image ).attr('src');
    output.html( "Image number " + this.currentIndex + " and href " + href );
  }
});
