oriel.js

simple javascript slideshow jquery extension

A simple demo.

Default options are used with only minimal styling in CSS. See the source →

HTML

<ul>
  <li>
    <img src="images/leaf.jpg">
  </li>
  <li>
    <img src="images/leaves.jpg">
  </li>
</ul>

JavaScript

$('ul').oriel();

CSS

.oriel_stage { /* Should always set a height. */
  height: 600px;
}
.oriel_navigation { /* Style the navigation bar. */
  padding: 0.2em;
  background-color: black;
  color: #aaa;
}
.oriel_navigation a {
  color: #ccc;
}

Oriel doesnâ€™t make any CSS changes with JavaScript by default. It swaps classes in and out instead. To make the images above fade in and out, the following CSS was used: 

.oriel-placeholder img { /* hidden by default */
  opacity: 0.0;
  position: absolute;
  transition: opacity 0.2s ease-in-out;
}
.oriel-placeholder img.oriel-active {
  opacity: 1.0;
}

With CSS3, you would only need to adjust the CSS to (for instance) have images slide in and out instead of fadeing.

Customizing.

The oriel object can control the slideshow programmatically.

For example: 

var el = $('ul').oriel(); // Create the slideshow
var slideshow = el.data('oriel'); // Obtain the slideshow object

slideshow.next(); // Go to next slide
slideshow.set(5); // Go to the 6th slide (0-indexed)
